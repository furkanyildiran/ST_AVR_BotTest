# AVRBotTest
NRF24L01 pins:  
PG0 : CE  
PG1 : CSN  
Left A4988 pins:  
EN(active low) : GND(always enabled)  
MS1 : PA5  
MS2 : PA7  
MS3 : PC6  
RST and SLP : PC4 with 10k series resistor  
STEP : PC2  
DIR : PC0  
Right A4988 pins:  
DIR : PA0  
STEP : PA2  
SLP and RST : PA4 with 10k series resistor  
MS3 : PA6  
MS2 : PC7  
MS1 : PC5  
EN(active low) : GND(always enabled)  
LCD pins:  
RST : PL7  
CE : PL6  
DC : PL5
